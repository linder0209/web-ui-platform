该文章中详细的介绍了选择器的针对性（specificity），即样式规则的优先级。大体规则如下：

style优先级最高

其次是id（#）

然后是选择器中其他属性和class的数量

最后是选择器中元素和伪类的数量

在同级别中，数量越大，优先级越高

比如：

h1 *[id=ok]   {}  /* a=0 b=0 c=1 d=1 -> specificity = 0,0,1,1  h1算元素，个数是1，[id=ok]伪类，个数也是1*/
ul ol li.red  {}  /* a=0 b=0 c=1 d=3 -> specificity = 0,0,1,3 ul ol li算元素，个数是3，.red算class，个数为1*/
li.red.level  {}  /* a=0 b=0 c=2 d=1 -> specificity = 0,0,2,1 */
