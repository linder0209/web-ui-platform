这里只简单的介绍在window环境下的安装与配置，在其他系统下安装以及详细介绍 参考这里

1、Node.js的安装

首先到官方http://nodejs.org/下载最近的版本，这里我们可以下载msi格式的安装文件，免去环境变量的配置，我们也可以到http://nodejs.org/dist/下载其他版本。下载后直接安装即可。

2、NPM的安装

NPM的全称是Node Package Manager。首先下载NPM源码，我们利用git（Git-1.8.0-preview20121022.exe）从github上下载，执行以下命令下载到指定目录下（假设clone到D盘的node目录下）

git clone --recursive git://github.com/isaacs/npm.git

下载成功后，会创建npm文件夹，进入该目录，执行如下命令，如果提示node: command not found，则需要确保正确设置了node.exe的环境变量path。如果安装在C盘下，需要设置path为C:\Program Files\nodejs，注意最后没有“\”。如果用Git-1.8.0-preview20121022.exe不能执行以下代码，则用cmd.exe，注意需要用管理员身份才能运行，即找到该文件所在的目录，右键该文件，以管理员身份运行。

cd D:\node\npm 
node cli.js install npm -gf
npm install underscore

分别运行完以上命令后，会返回以下内容

underscore@1.4.3 node_modules\underscore

说明运行成功。
