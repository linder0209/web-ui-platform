Array中的sort方法

Array.prototype.sort方法是对数组进行排序的。该方法是在原数组上排序，不是返回副本。如果调用方法sort()时没有参数，将按照字母排序（更为精确地说，是按照字符编码的顺序）对数组中的元素排序的。如果有参数（该参数是一比较函数）时，会根据该比较函数返回值来排序。比较函数有两个参数a和b，其返回值如下：

如果a小于b，在排序结果中想让a出现在b之前，则需要返回一个小于0的值，即正序排序

如果a等于b，就返回0

如果a大于b，返回大于0的值

以上是正序排序，如果是倒序则按反向返回数值

注意，数组中的undefined的元素都会排在数组末尾，即使提供了自定义的排序函数，也是这样，因为undefined值不会被传递给你提供的比较函数。

例子：

function numberRorder(a,b){
    return a- b;
}
var a = [33,4,1111,222];
a.sort();// 1111,222,33,4
a.sort(numberRorder);//4,33,222,1111

对多维数组的排序

var arr = [
    [2, 1, 55, 4],
    [5, 3, 22, 3],
    [1, 2, 77, 2],
    [9, 4, 33, 5]
];
console.info("默认按第一列排\n" + arr.sort().join("\n"));
console.info("现在按第三列排\n" + arr.sort(function(left, right) {
    return left[2] > right[2] ? 1 : -1;
}).join("\n"));
console.info("现在按第三列倒排\n" + arr.sort(function(left, right) {
    return left[2] > right[2] ? -1 : 1;
}).join("\n"));
