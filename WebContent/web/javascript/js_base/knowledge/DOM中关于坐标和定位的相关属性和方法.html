window对象中坐标和位置的相关属性

包含以下几个属性

浏览器距离屏幕y坐标值：window.screenTop ，firefox中用window.screen.top代替

浏览器距离屏幕x坐标值：window.screenLeft ，firefox中用window.screen.left代替 

屏幕的高度：window.screen.height  

屏幕的宽度：window.screen.width  

屏幕可见工作区y坐标：window.screen.availTop

屏幕可见工作区x坐标：window.screen.availLeft

屏幕可用工作区高度：window.screen.availHeight  

屏幕可用工作区宽度：window.screen.availWidth

看下面的测试代码

console.log(window.screenTop);//距离屏幕y坐标值，firefox不支持该属性
console.log(window.screenLeft); //距离屏幕x坐标值，firefox不支持该属性
console.log(window.screen.top);//firefox中用该属性代替window.screenTop
console.log(window.screen.left);//firefox中用该属性代替window.screenLeft
console.log(window.screen.height); //屏幕的高度
console.log(window.screen.width); //屏幕的宽度
console.log(window.screen.availTop); //屏幕可见工作区y坐标
console.log(window.screen.availLeft); //屏幕可见工作区x坐标
console.log(window.screen.availHeight); //屏幕可见工作区高度，即除去任务栏的高度
console.log(window.screen.availWidth);//屏幕可见工作区宽度，即除去任务栏的宽度

document对象中坐标和位置的相关属性

包含以下几个属性

浏览器距离屏幕y坐标值：document.body.clientWidth

浏览器距离屏幕x坐标值：document.body.clientHeight 

屏幕的高度：window.screen.height  

屏幕的宽度：window.screen.width  

屏幕可见工作区y坐标：window.screen.availTop

屏幕可见工作区x坐标：window.screen.availLeft

屏幕可用工作区高度：window.screen.availHeight  

屏幕可用工作区宽度：window.screen.availWidth

看下面的测试代码

console.log(document.body.clientWidth);//页面可视宽度，不包括滚动条
console.log(document.body.clientHeight);//页面可视高度，不包括滚动条
console.log(document.body.offsetWidth);//body实际宽度，不包括滚动条和外边距margin，但包括边框和内边距
console.log(document.body.offsetHeight);//body实际高度，同上
console.log(document.body.scrollWidth);//页面实际宽度，不包括滚动条
console.log(document.body.scrollHeight);//页面实际高度，不包括滚动条
console.log(document.body.scrollTop);//竖向滚动条的坐标
console.log(document.body.scrollLeft);//横向滚动条的坐标

我们分别在各种文档模式（DOCTYPE，或者称为Document Type Declaration（文档类型声明，缩写 DTD））和常见浏览器下测试。其中文档模式选用以下几种：

隐式（即不设置文档模式）

Frameset模式：<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

Strict模式：<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN" "http://www.w3.org/TR/html4/strict.dtd">

Transitional模式：<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

html5模式：<!DOCTYPE html>

常用浏览器选用Firefox、Chrome和万恶的IE（尤其IE6和7）
